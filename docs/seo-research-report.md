## 2. SEO強み（現状）

### 2-1. エリアSEO網羅性（最大の強み）

横浜市18区すべてにエリアページを設置済み。さらに区内の主要駅・地区・路線別ページを作成しており、エリア × 地区のロングテールキーワードをカバーする構造が整っている。

- 横浜市全18区: 鶴見区・神奈川区・西区・中区・南区・港南区・保土ケ谷区・旭区・磯子区・金沢区・港北区・緑区・青葉区・都筑区・戸塚区・栄区・泉区・瀬谷区
- 川崎市・厚木市・海老名市にも対応ページあり
- 「[区名] 給湯器交換」「[駅名] 給湯器交換」での上位表示を狙える構造

### 2-2. 症状別トラブルページ（9症状）

ユーザーが検索する症状・トラブルキーワードに対応したランディングページ群。

対象症状例:
- お湯が出ない
- 水漏れ
- エラーコード表示
- 給湯器から変な音がする
- お湯の温度が安定しない
- 追い焚きができない
- 点火しない
- 熱いお湯が出ない
- 給湯器の交換時期の判断

症状別ページは検索意図が明確で、CVR（コンバージョン率）が高くなる傾向がある。

### 2-3. 施工事例45件（全件FAQPage Schema付き）

45件の施工事例ページはすべてFAQPage SchemaおよびBreadcrumbList Schemaを実装済み。実績の積み上げがE-E-ATに直結する。施工前後の写真・工事内容・費用感を掲載することで、ユーザーの不安解消にも機能する。

### 2-4. ガイドページ12件（expertise確立）

給湯器に関する専門的な解説ガイドを12件掲載。E-E-A-T（経験・専門性・権威性・信頼性）の基礎として機能する。

ガイド例（推定）:
- 給湯器の選び方
- 号数の選び方
- エコジョーズとは
- 交換費用の相場
- 寿命・交換時期の目安
- 工事の流れ
- メーカー別解説

### 2-5. 全ページSchema実装済み

全ページに以下のSchema.orgを実装済み:
- `LocalBusiness`（会社情報・エリア情報）
- `BreadcrumbList`（階層構造の明示）
- `FAQPage`（よくある質問形式のコンテンツ）

これにより、Googleのリッチリザルト（FAQリッチスニペット等）表示の可能性がある。

### 2-6. Core Web Vitals対応済み

Next.js 14（App Router）を使用しており、以下の最適化が標準で機能している:
- `next/image` による画像最適化・Lazy Load
- サーバーサイドレンダリング（SSR）/ 静的生成（SSG）
- フォント最適化（`next/font`）
- Code Splitting・Dynamic Import

## 3. SEO弱み・改善余地

### 3-1. ブログが未充実（最優先課題）

現状ブログ記事数: 約10件（推定）

問題点:
- 記事数が少なすぎて、ロングテールキーワードの網羅ができていない
- 被リンクを獲得する「引用されるコンテンツ」がない
- E-E-A-T向上のための定期的なコンテンツ更新が不足
- AIによる検索結果への引用機会を逃している

目標: 6ヶ月で120記事、1年で200記事

### 3-2. 外部被リンクがほぼゼロ

推定状況: 被リンク数が非常に少ない（ドメイン権威が低い状態）

問題点:
- Googleが「権威あるサイト」と判断する根拠が薄い
- 競合他社（大手給湯器業者・リフォーム会社）に比べてドメインオーソリティが劣る
- ローカルSEOでもサイテーション（NAP情報の引用）が不足している可能性

改善方向:
- 業者ディレクトリへの登録（ミツモア・生活110番等）
- コンテンツマーケティングによる自然な被リンク獲得
- 地域パートナーとの紹介連携

### 3-3. GBP（Googleビジネスプロフィール）の状態が不明

要確認事項:
- 写真枚数（目標: 100枚以上）
- 口コミ数（目標: 50件以上）
- 投稿頻度（目標: 週1〜2回）
- GBPのインサイト数値（インプレッション・クリック数）

GBPはローカルSEOにおける最重要シグナルであり、横浜市エリアでの「給湯器交換 横浜」検索での表示に直結する。

### 3-4. 競合比較ページがない

潜在顧客が宝宮設備を選ぶ際に比較検討する、「なぜ宝宮設備を選ぶべきか」を訴求するページが不足している。

必要なコンテンツ:
- 他業者との比較表（価格・対応スピード・保証等）
- 「大手ガス会社に頼む場合 vs 専門業者に頼む場合」の比較
- 口コミ・実績を活かした信頼訴求

### 3-5. 価格訴求ページが薄い

「給湯器 交換 費用」「給湯器 価格 相場」は検索ボリュームが非常に高いキーワード群だが、現状ではこれに特化した充実したページがない。

必要なコンテンツ:
- 給湯器交換費用の詳細相場ページ
- 号数・タイプ別の料金表
- 見積もりの読み方ガイド
- 「なぜ安いのか」「なぜ高いのか」を透明に説明するページ

## 4. 競合ギャップ分析

### 4-1. 横浜エリアの主な競合傾向

地域密着型の給湯器交換業者・リフォーム会社は横浜エリアのSEO対策として以下を実施している傾向がある:
- GBPの口コミ数が多い（50件以上）
- 費用相場ページの充実
- ブログ記事の定期投稿（月10〜30記事）
- 大手ポータルサイト（ミツモア・ホームプロ等）への掲載

### 4-2. 当サイトの差別化点（現在の優位性）

宝宮設備サイトの三位一体構造が競合に対する優位性を生んでいる:

```
エリアページ（90+ページ）
      ×
症状別トラブルページ（9ページ）
      ×
施工事例（45件）
```

この掛け合わせにより、「[区名] 給湯器 水漏れ」「[駅名] 給湯器 エラーコード」といった複合キーワードでの上位表示が狙える。大手競合が持たない細粒度のエリアカバレッジが最大の強み。

### 4-3. 競合に勝てる領域

1. **ロングテールキーワード（区 × 症状 × メーカーの掛け合わせ）**
   - 例: 「港北区 リンナイ 給湯器 エラーコード 修理」
   - 例: 「青葉区 エコジョーズ 交換 費用」
   - 例: 「鶴見区 給湯器 水漏れ 急ぎ」
   - 競合する月間検索数は少ないが、CVRが高く、かつ競合が少ない

2. **施工事例コンテンツの蓄積**
   - 事例数45件は中規模業者として多い部類
   - 事例数を60〜100件に増やすことで、検索結果での露出と信頼性がさらに向上

3. **ガイドコンテンツの深掘り**
   - 既存12件のガイドをさらに拡充し、「給湯器 選び方 横浜」等での上位表示を目指す

### 4-4. 競合が強い領域（当面は注意）

- 「給湯器交換 横浜市」「給湯器 交換 費用 神奈川」等のビッグキーワード
- ドメインオーソリティが高い大手リフォームポータル（ホームプロ・リフォームの窓口等）との競合
- GBPのMEOパック（上位3件のローカルパック表示）での激戦

## 5. 優先改善アクション TOP10

以下を優先度順に実施する。

### 優先度A（即時実施: 0〜1ヶ月）

**1. GBP完全最適化**
- 写真を月8枚以上投稿し、累計50枚以上を目指す
- 口コミ依頼フローを施工完了時に必ず実施
- GBP投稿を週1〜2回に増やす
- 説明文をキーワードを含む500文字に最適化

**2. Search Console設定・監視強化**
- 主要キーワード20件の順位を週次でモニタリング
- クロールエラーの定期確認
- sitemap.xmlの再送信・確認

**3. ブログ投稿開始**
- 月20記事のペースで投稿開始
- 最初の優先テーマ: エラーコード解説・給湯器の選び方
- 各記事1000文字以上・FAQセクション付き

### 優先度B（1〜3ヶ月以内）

**4. 費用・相場ページの充実**
- 給湯器交換費用相場の詳細ページを新設または大幅強化
- 号数別・タイプ別の価格表を追加
- 「見積もりの見方」ガイド追加

**5. 業者ディレクトリへのNAP登録**
- ミツモア・生活110番・ホームプロへのプロフィール登録・整備
- Yahoo!ロコ・Bing Placesへの登録確認

**6. 施工事例の月次追加（3件/月）**
- 45件から6ヶ月後に60件超を目指す
- 地域・機種・症状のバリエーションを意識して追加

### 優先度C（3〜6ヶ月以内）

**7. 競合比較・選ばれる理由ページの新設**
- 「なぜ宝宮設備が選ばれるのか」を数値と実績で訴求するページ
- 大手との比較コンテンツ

**8. E-E-A-T強化（会社情報・スタッフ情報の充実）**
- 代表者プロフィールの充実（資格・経験年数）
- 資格・認定情報の構造化データ追加

**9. 内部リンク最適化**
- ブログ記事からエリアページ・症状ページへの内部リンクを体系化
- 施工事例からガイドページへの導線強化

**10. Core Web Vitals計測・改善**
- Lighthouse・PageSpeed Insightsでの定期計測
- 画像最適化・キャッシュ設定の見直し

## 7. 補足・注意事項

- 本レポートは2026年6月時点の推定に基づく。Search Consoleの実際のデータを確認して数値を補正すること。
- SEOの効果発現には通常3〜6ヶ月かかる。短期間での劇的な改善を期待せず、継続的な施策実行を最優先とする。
- Googleのアルゴリズム変更に備え、コンテンツの品質向上とユーザー体験の改善を根幹施策とする。
- 黒hat SEO（リンク購入・キーワードスタッフィング・コピーコンテンツ等）は絶対に実施しない。
