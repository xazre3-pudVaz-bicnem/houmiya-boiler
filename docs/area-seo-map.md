## 内部リンク構造

```
/area                                    ← ハブページ
├── /area/yokohama                       ← 横浜市トップ（最重要）
│   ├── /area/yokohama/[ward]           ← 18区ページ
│   │   ├── /area/yokohama/station/[s] ← 駅ページ（区から参照）
│   │   └── /area/yokohama/district/[d]← 地域ページ（区から参照）
│   ├── /area/yokohama/trouble/[t]      ← 症状19件
│   ├── /area/yokohama/type/[t]         ← タイプ18件
│   ├── /area/yokohama/maker/[m]        ← メーカー3件
│   ├── /area/yokohama/capacity/[c]     ← 号数3件
│   └── /area/yokohama/building/[b]     ← 建物5件
├── /area/kawasaki
│   └── /area/kawasaki/[ward]           ← 7区ページ
├── /area/atsugi                         ← 本社所在地
└── /area/ebina
```
