## 1. 戦略概要

### 1-1. ブログの目的

1. **E-E-A-T（経験・専門性・権威性・信頼性）の向上**
   - 専門的な知識を発信し続けることで、Googleに「給湯器の専門家」として認識させる
   - 著者情報・資格情報を明示し、信頼性シグナルを積み上げる

2. **ロングテールキーワードの獲得**
   - 「エラーコード632 ノーリツ」「給湯器 16号 20号 どっち」等の具体的な検索クエリに対応
   - 月間検索数が少なくても検索意図が明確なキーワードで確実に流入を獲得する

3. **被リンク・引用の獲得**
   - 専門性・網羅性の高い記事が他サイトから自然にリンクされる素地を作る
   - ChatGPT・Perplexity・Geminiが質問に回答する際に当サイトのコンテンツを引用することを狙う

4. **内部リンク強化**
   - ブログ記事からエリアページ・症状ページ・施工事例へ内部リンクを設置し、サイト全体の評価を高める

### 1-2. 投稿ペース

- 月20記事を6ヶ月継続 = 120記事
- 7〜10ヶ月目に残り80記事を投稿 = 合計200記事
- 1記事あたりの最低品質基準: 1,000文字以上・FAQセクション必須・内部リンク3本以上

## 4. 記事品質基準

### 4-1. 文字数・構成

- 最低1,000文字以上（推奨: 1,500〜2,000文字）
- H1タグ: 記事タイトル
- H2タグ: 3〜5個の大見出し
- H3タグ: 必要に応じて小見出し
- FAQセクション: 必須（H2「よくある質問」として最低3問）
- まとめセクション: 推奨（CTAを含める）

### 4-2. SEO最適化

- titleタグ: メインキーワードを含む60文字以内
- descriptionタグ: 120文字以内・検索意図に沿った説明
- FAQPage Schema: 全記事に実装
- BreadcrumbList Schema: 全記事に実装
- canonical URL: 必ず設定

### 4-3. 内部リンク

- 各記事から最低3本の内部リンクを設置
- 優先リンク先: エリアページ（/area/）・症状ページ（/trouble/）・施工事例（/cases/）
- アンカーテキストはキーワードを自然に含める

### 4-4. コンテンツ品質

- 事実確認済みの情報のみ掲載
- 独自の知見・経験を含める（施工現場の写真等）
- 読者の「なぜ？」「どうやって？」に具体的に答える
- AI生成のみに頼らず、専門家の知見を加える

### 4-5. CTA（行動喚起）

- 各記事の末尾に「お問い合わせはこちら」「無料見積もり」等のCTAを設置
- 電話番号・LINE・フォームへの導線を記事内に含める
