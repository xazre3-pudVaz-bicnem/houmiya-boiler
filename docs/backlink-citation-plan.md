## 1. 基本方針

### 1-1. 方針の概要

外部SEO施策は「自然な被リンク・引用のみ」を原則とする。

**絶対に行わないこと（ペナルティリスク）**:
- リンク購入（有料での被リンク設置）
- スパムディレクトリへの大量登録
- 自動投稿ツールによる大量サイテーション生成
- 相互リンクの過剰実施（SEO目的のみの相互リンク）
- 記事スパム（品質の低い記事を大量に他サイトへ投稿）
- コメントスパム（ブログのコメント欄にリンクを貼る行為）
- フォーラム・掲示板への無関係なリンク投稿

**推進すること**:
- 正確なNAP情報の主要ディレクトリへの登録（サイテーション構築）
- 専門性の高いコンテンツによる自然なリンク獲得
- 地域パートナーとの信頼関係を基にした協力関係
- SNSを通じた認知拡大（間接的なシグナル）

### 1-2. 被リンクよりサイテーションを優先する理由

地域ビジネス（ローカルSEO）においては、被リンク数よりも「NAP情報の一貫性と引用数（サイテーション）」がローカルパック表示に大きく影響する。まず正確なNAP情報をできるだけ多くの信頼できるサイトに掲載することを最優先とする。

**チェックリスト**
- [ ] すべてのプラットフォームでNAPが完全に一致しているか確認
- [ ] Webサイトのフッター・会社情報ページのNAPと一致しているか
- [ ] 電話番号のハイフン有無が統一されているか

これらは信頼性の高いドメインからのリンク・サイテーションとなるため、ドメインオーソリティ向上に寄与する。

## 3. コンテンツ主導の被リンク獲得

### 3-1. 引用されやすいコンテンツの制作（リンクベイト）

意図的にリンクやSNSシェアを獲得しやすいコンテンツを作成する。

**タイプA: 網羅性の高いガイドコンテンツ**

既存の /guide/ ページをさらに詳細に拡充する。他のブログや情報サイトが「詳しくはこちら」と引用したくなる内容にする。

- 給湯器のメーカー・型番・価格の完全比較表（定期更新）
- 横浜市の給湯器交換費用相場の年次調査レポート
- 給湯器エラーコード全機種対応一覧（リンナイ・ノーリツ・パロマ）
- 給湯器の号数選択シミュレーター（家族人数・使用量から計算）

**タイプB: 地域特化の情報**

横浜・神奈川エリアのユーザー向けに特化した情報は競合が少なく、引用されやすい。

- 横浜市の給湯器交換補助金・助成金情報（市の公式情報をまとめた記事）
- 神奈川県の給湯器対応業者マップ（自社を含む）
- 横浜市エリア別の給湯器トラブル頻度の傾向レポート

**タイプC: 専門知識・FAQコンテンツ**

ChatGPT・Perplexity等のAI検索が引用しやすいQ&A形式のコンテンツ。

- 「給湯器FAQ 100問100答」のような大型コンテンツ
- エラーコード別の対処法データベースページ
- 給湯器メーカー別の型番早見表

### 3-2. プレスリリース（地域メディア向け）

以下のような事業活動をプレスリリース化して地域メディアに配信する。

**プレスリリースの機会例**:
- 創業○周年記念（節目のタイミング）
- 施工実績100件達成（マイルストーン）
- 新しいサービス開始（写真見積もりサービス等）

- 冬の凍結対策キャンペーン（季節性のニュース）

**配信先（無料・低コスト）**:
- PR TIMES（prtimes.jp）: 有料だが信頼性が高い
- AtPress（atpress.ne.jp）: 無料プランあり
- ValuePress（value-press.com）: 無料プランあり

**地域メディアへの直接提案**:
- カナコム（神奈川県内ニュースサイト）
- 横浜ウォーカー（地域情報サイト）
- 神奈川新聞の地域欄
- 横浜市内の各区の地域情報ブログ・メディア

## 4. パートナーシップ（相互紹介・連携）

### 4-1. 基本方針

SEO目的の相互リンクではなく、実際のビジネス連携の結果として自然にリンクが生まれることを目指す。

### 4-2. パートナー候補業種と連携方法

**不動産会社・賃貸管理会社**

連携内容:
- 入居者向けの給湯器故障時の修理業者として紹介してもらう
- 退去・リフォーム時の給湯器交換業者として紹介してもらう
- 管理会社のWebサイトや案内文書に宝宮設備の情報を掲載してもらう

アプローチ方法:
- 横浜市・川崎市内の管理会社に営業・挨拶訪問
- 「緊急対応可能な給湯器専門業者」として契約業者リストへの登録を依頼

**リフォーム会社・内装業者**

連携内容:
- リフォーム工事の一環として給湯器交換が発生した場合の紹介
- 浴室リフォーム時の給湯器交換を担当
- お互いのWebサイト・SNSで相互紹介（ただしSEO目的ではなくビジネス連携として）

**ハウスメーカー・工務店**

連携内容:
- 施工後のアフターサービス・給湯器交換部門としての協力
- 新築・リノベーション物件の給湯器工事担当として参画

**電気工事・水道工事業者**

連携内容:
- 電気工事・水道工事で訪問した際に給湯器の状態が悪い場合に紹介してもらう
- 逆に給湯器工事中に電気・水道の問題が見つかった場合に紹介する相互紹介体制

**ガス会社・ガス販売会社**

連携内容:
- ガス会社の紹介業者リストへの登録を依頼
- プロパンガス販売会社からの給湯器交換需要を紹介してもらう

### 4-3. パートナー管理台帳

## 5. SNS活用（間接的な認知・シグナル）

### 5-1. SNSとSEOの関係

SNSのフォロワー数・いいね数は直接的なランキング要因ではないが、以下の間接効果がある:

- コンテンツが拡散されることで被リンクを獲得する機会が増える
- ブランド認知が高まることでブランド名での直接検索が増える（GBPシグナル向上）
- E-E-A-TにおけるAuthority（権威性）の補助的な指標になる

### 5-2. 各SNSの活用方針

**X（旧Twitter）**

投稿内容:
- エラーコード別の対処法ミニ情報（「エラー111が出たら、まず○○を確認」）
- 季節に合わせたトラブル予防情報（「今日は気温が急低下。凍結対策を確認してください」）
- 施工事例の紹介（ビフォーアフター写真）
- ブログ新記事の告知

投稿頻度: 週2〜3回
目標: 専門情報アカウントとして認知される・他業界アカウントからの引用を狙う

**Instagram**

投稿内容:
- 施工ビフォーアフター写真（キレイに仕上がった写真）
- 作業風景の写真・動画
- 「給湯器の正しい使い方」等のインフォグラフィック

ハッシュタグ: #給湯器交換 #横浜市 #神奈川 #エコジョーズ #給湯器 #リンナイ #ノーリツ #給湯器工事

投稿頻度: 週1〜2回

**YouTube（中長期施策・優先度低）**

動画コンテンツ案:
- 「給湯器交換の工事の流れを実際の現場で解説」
- 「号数の選び方を3分で解説」
- 「エラーコード別の対処法シリーズ」
- 「ビフォーアフター施工事例紹介シリーズ」

YouTubeは競合が少なく、長期的に検索トラフィックを獲得できる。給湯器関連のYouTube検索は一定の需要がある。

投稿頻度（開始後）: 月2本を目標
※専任担当・機材が準備できるタイミングで開始する

**LINE公式アカウント**

活用目的: SEO効果は薄いが、顧客との継続的な関係構築・口コミ依頼に有効

- 施工完了後の口コミ依頼メッセージ配信
- 季節ごとのキャンペーン案内
- 新サービス・ブログ更新の通知

## 6. 禁止事項リスト（再確認）

以下は絶対に実施しない。発覚した場合はGoogleのペナルティ（検索順位の大幅降下・インデックス除外）を受けるリスクがある。

| 禁止行為 | 理由 |
|---|---|
| リンク販売サービスへの依頼・購入 | Googleのリンクスパムポリシー違反 |
| 有料ディレクトリへの大量登録（SEO目的） | スパムディレクトリへのリンクはマイナス評価の可能性 |
| 記事スパム（低品質記事を大量投稿して被リンクを獲得） | Google品質ガイドライン違反 |
| ブログコメント・フォーラムへのスパムリンク | ガイドライン違反・サイトの評判低下 |
| プレスリリース配信でのアンカーテキストリンクの大量設置 | 不自然なリンクパターン |
| 相互リンクの過剰実施（SEO目的のみ） | リンクスキーム認定のリスク |
| 無関係なサイトへの大量ゲスト投稿（リンク目的） | ガイドライン違反 |

## 7. 月次進捗管理

### 7-1. Search Consoleによる被リンクモニタリング

Google Search Console の「リンク」レポートで以下を月次確認する:

- 外部リンク数（総数）の推移
- リンク元サイトの一覧（新規リンク元の確認）
- リンクされている自サイトのURL（どのページがリンクされているか）
- 不自然なリンクの有無（スパムサイトからのリンクは否認を検討）

### 7-2. 月次KPI

| 指標 | 確認方法 | 目標（6ヶ月後） |
|---|---|---|
| 外部被リンク総数 | Search Console > リンク | 現状 + 30件以上 |
| リンク元ドメイン数 | Search Console > リンク | 現状 + 20ドメイン以上 |
| ディレクトリ登録数 | 台帳管理 | 10サイト以上 |
| パートナー企業数 | 台帳管理 | 3社以上 |
| SNSフォロワー数（X） | Twitter Analytics | 200以上 |
| SNS投稿数（合計） | 各プラットフォーム | 月20投稿以上 |

### 7-3. スパムリンクの否認

Search Consoleのリンクレポートで明らかにスパム性の高いサイトからのリンクを発見した場合:

1. 否認対象リンクのURLリストをテキストファイルで作成
2. Search Console の「リンクの否認」ツールでアップロード
3. 月次確認の際に否認ファイルを更新

否認は慎重に行う。正規サイトからのリンクを誤って否認するとSEOにマイナスになる。
